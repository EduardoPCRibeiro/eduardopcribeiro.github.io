Deriving Sound Speed from Open-Source CTD Profiles

Tools: Python • PostgreSQL • QGIS • Multiple Sound-Speed Equations

Goal

Process large CTD datasets, clean them via SQL, compute sound speed using multiple equations (UNESCO, Del Grosso, Leroy, Mackenzie), and generate geospatial visualizations.

Workflow

1. Data Cleaning in PostgreSQL

Raw CTD files were reshaped and cleaned in Postgres.

2. Projection to UTM


from pyproj import Transformer

origin_crs = "EPSG:4326"
target_crs = "EPSG:32611"

transformer = Transformer.from_crs(origin_crs, target_crs, always_xy=True)
e, n = transformer.transform(lon, lat)
